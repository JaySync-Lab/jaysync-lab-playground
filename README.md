# jaysync-lab-playground

Working repo for the JaySync-Lab Interactive Playground — an isolated,
throwaway sandbox environment for hands-on infrastructure practice.

This repo holds the in-progress implementation log, controller code, and
scripts as they're built phase by phase. Once a phase is stable, a polished
writeup gets added to [`JaySync-Lab/docs/playground`](https://github.com/JaySync-Lab/JaySync-Lab)
— that's the version meant for public reading. This repo is the working log,
not the final word.

**Current status:** Phase 1 (network foundation) and Phase 2 (golden
template, CT 180) complete and verified. Phase 3 (session controller) next.

See [`implementation-log.md`](./implementation-log.md) for full detail.
